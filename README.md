## 📋 项目概述

本项目是一个完整的端到端（End-to-End）金融数据工程与分析实战项目。项目旨在通过编写自动化 Python 脚本，从互联网动态抓取中国 A 股市场（沪、深、北三大交易所）**全部在市**公司的全量基础名录，并通过内置的特征降维算法清洗重塑数据，最后基于清洗后的高质量数据展开多维度的探索性数据分析（EDA）与可视化。

**数据规模**: 
- 时间跨度：1990年至今（覆盖 A 股三十余年历史）
- 样本量：约 5,400+ 家在市公司
- 覆盖板块：沪深主板、创业板、科创板、北交所

---

## 🎯 两大核心任务概览

### Task A: 数据自动化采集与机构级清洗重塑
**执行脚本**: `01_data_clean.ipynb`  
**目标**: 摆脱本地静态文件依赖，实时在线拉取三大交易所最新公司名录，并进行底层数据特征重构。

**核心清洗工程**:
1. **格式规范化**: 自动为股票代码追加标准化后缀（`.SZ`, `.SH`, `.BJ`），精准推断并统一上市板块名称。
2. **行业降维映射**: 针对原生网页接口行业极其细碎（上百种）的问题，内置**特征分类矩阵**，利用 NLP 规则将所有公司精准收敛降维至经典的 11 个宏观一级行业（如金融、信息技术、工业等）。

### Task B: 多维度探索性数据分析 (EDA) 与可视化
**执行脚本**: `02_China_stock_analysis.ipynb`  
**目标**: 基于 Task A 产出的干净数据，从四大宏观视角透视中国资本市场格局，并自动导出高清图表。

## 📌 核心分析结论

基于本项目生成的数据探索与可视化图表，得出以下四大宏观洞察：

- **政策驱动特征显著**: 制度创新（如注册制试点、北交所设立）是激发市场活力的最核心引擎，各年度 IPO 数量呈现明显的政策周期特征。
- **多层次板块新老交替**: 沪深主板沉淀了大量上市超 15 年的老牌企业；而科创板与北交所作为“新鲜血液”，大幅拉低了市场整体年龄中位数，体现了资本市场对硬科技与创新型中小企业的精准滴灌。
- **“硬科技”与“高端制造”筑基**: 经降维分析发现，“工业”与“信息技术”已成为当前 A 股容纳企业数量最大的两个超级板块，深度契合国家向高端制造业转型的宏观战略。
- **三大交易所错位竞争格局**: 上交所大盘与科创属性并重；深交所受创业板带动，“新经济”与信息技术色彩浓厚；北交所则高度聚焦“专精特新”，工业企业占比呈现压倒性优势。

## 💡 使用指南

### 查看分析结果
1. **完整代码和逻辑**: 打开 `.ipynb` 查看完整的 Jupyter Notebook 运行过程及代码解释。
2. **清洗后数据集**: 进入 `data_clean` 文件夹查看格式化后的标准 CSV 宽表文件。
3. **可视化图表**: 所有的统计图表均已自动按 300dpi 高清导出至 `output` 文件夹中，可直接插入课题演示文稿。

### 重现分析流程
```bash

pip install pandas numpy matplotlib seaborn akshare

# 3. 运行图表渲染与 EDA 脚本（自动生成 output 目录及高清图表）
jupyter notebook 02_China_stock_analysis.ipynb
```
