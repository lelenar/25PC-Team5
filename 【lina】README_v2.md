## 📋 项目概述

本项目是一个完整的端到端（End-to-End）金融数据工程与量化分析实战项目。项目旨在通过编写自动化 Python 脚本，从互联网动态抓取中国 A 股市场（沪、深、北三大交易所）**全部在市**公司的全量基础名录，并通过构建高可用的特征降维算法清洗重构数据，最后基于清洗后的高质量数据展开多维度的探索性数据分析（EDA）与可视化。

**核心数据规模**: 
- **时间跨度**：1990年（深圳“老八股”上市）至 2026 年（覆盖 A 股 37 年历史）
- **样本体量**：全量 5,490 家在市公司
- **覆盖板块**：沪深主板、创业板、科创板、北交所

---

## 🎯 核心工程与分析模块

### 模块一：数据自动化采集与机构级清洗重塑 (`01_data_craw_and_clean.ipynb`)
摆脱本地静态文件依赖，实时在线拉取三大交易所最新公司名录，并进行底层数据特征重构。

- **双源互补机制**：创新性引入“东方财富 (akshare) + Baostock”双源行业字典。有效解决了单一数据源缺失的问题，成功还原了北交所 299 家“专精特新”企业的真实行业面貌。
- **格式规范化**：自动为股票代码追加标准化后缀（`.SZ`, `.SH`, `.BJ`），精准推断并统一上市板块名称。
- **行业降维映射**：针对原生网页接口行业极其细碎（90+ 种细分行业）的问题，内置严谨的 **NLP 关键词分类矩阵**，将全市场公司精准收敛降维至**中证（CSI）标准的 11 个宏观一级行业**。

### 模块二：宏观视角下的探索性数据分析 (EDA) (`02_China_stock_analysis.ipynb`)
基于阶段一输出的高质量清洗宽表，跨越横向（空间/行业）与纵向（时间/周期）维度，输出 4 大分析视角与精美可视化图表：

1. **政策干预的量级影响**：揭示 2013 年 IPO 暂停（断崖式下跌 97%）与 2021 年注册制落地（新增 524 家创历史峰值）等重大政策节点对资本市场扩容的绝对驱动力。
2. **多层次市场的世代交替**：量化展现 A 股“十年一代”的扩容引擎切换（主板 -> 创业板 -> 科创板/北交所），以及各板块精确递减的“5年阶梯”生命周期中位数。
3. **“硬科技”与“高端制造”筑基**：“工业”（29.9%）与“信息技术”（20.8%）成为当前 A 股容纳企业数量最大的两个超级板块，深度契合国家向高端制造业转型与数字经济发展的宏观战略。
4. **三大交易所错位竞争格局**：上交所呈现“压舱石+硬科技”双轨并重；深交所“新经济”与消费色彩浓厚；北交所则高度聚焦“专精特新”，呈现出极强的板块差异化生态。

## 📌 核心分析结论与宏观洞察

本项目基于跨越 37 年（1990-2026）的长周期全量数据，深度穿透了中国资本市场的发展脉络，提炼出以下三大核心量化发现：

**1. 政策驱动与周期印记：干预量级远超自然市场周期**
制度创新（如注册制试点、北交所设立）是激发资本市场活力的最核心引擎。量化数据显示，政策节点对市场扩容具有绝对驱动力：2013 年的 IPO 暂停导致全年新增断崖式下跌 **97%**（仅约 2 家）；而 2021 年注册制红利的集中释放，促使全年新增上市达 **524 家**，创下 A 股 37 年历史峰值。同时，宏观制度的落地通常存在 **1—2 年的滞后蓄水期**。

**2. 多层次市场交替：“十年一代”与“历史时间轴”的精准映射**
A 股扩容引擎经历了清晰的代际切换：沪深主板（1990-2003） → 创业板（2004-2018） → 科创板/北交所双轨（2019 至今）。截面数据显示，沪深主板沉淀了大量老牌企业，而科创板与北交所作为“新鲜血液”，形成了极为清晰的梯次递减结构：**主板（15.8年） → 创业板（7.5年） → 科创板（4.7年） → 北交所（3.2年）**。这一阶梯特征绝非数字巧合，而是完美对应了我国多层次资本市场设立与扩容的历史时间节点，深刻体现了资本市场“梯次开放、分层服务”的顶层设计：年龄越小的板块，服务的企业越偏向于新一代硬科技与创新型中小企业。

**3. 产业底色重构：“实体制造”筑基与“数字科技”驱动**
经中证 11 大行业降维清洗，A 股呈现出清晰的国家战略底色。**工业（29.9%）**与**原材料（17.7%）**合计占比近半，构筑了不可动摇的“制造强国”基本盘；**信息技术（20.8%）**稳居第二大行业，这不仅深度契合国家向高端制造业转型的宏观战略，更反映出资本市场对数字经济底层基础设施的强力赋能。

**4. 三大交易所错位竞争格局**：上交所呈现“压舱石 + 硬科技”双轨并重，行业分布最为多元与均衡；深交所突出“新制造 + 新经济”，在高端制造与数字经济领域的聚集效应极强；北交所则高度聚焦“专精特新”，工业占比（42.5%）呈现压倒性优势，下沉服务生态极度纯粹化。

## 💡 使用指南

### 1. 运行环境配置
请确保本地 Python 环境已安装以下必需的第三方依赖库：
```bash
pip install pandas numpy matplotlib seaborn akshare baostock openpyxl
```

### 2. 重现分析流程
本项目分为两个相互解耦的执行阶段，支持全流程一键复现：

第一步（数据管道）: 运行 01_data_craw_and_clean.ipynb。该脚本需保持网络畅通，将自动完成三大交易所数据全量爬取与双源字典清洗，并在 data_clean 目录下生成标准化 CSV 宽表。

第二步（分析引擎）: 运行 02_China_stock_analysis.ipynb。该脚本将直接读取清洗后的宽表，执行数据透视与深度分析，并在 output 文件夹中按 300dpi 高清导出所有可视化图表。

启动 Jupyter 环境，依次打开并运行上述两个脚本

### 3. 查看分析成果
完整代码与逻辑: 打开对应的 .ipynb 文件，查看带有详尽 Markdown 标注的执行过程、算法解释及业务解读。

采集的原始数据集：进入 data_raw 文件夹，查看采集后的 CSV 宽表文件（china_listed_official_data.csv）。

清洗后数据集: 进入 data_clean 文件夹，查看格式化后的标准 CSV 宽表文件（china_listed_official_data_patched.csv）。

可视化图表: 所有的探索性数据分析（EDA）图表均已自动导出至 output 文件夹。
